# Manuel Utilisateur : Opérateur de Vente (Guichetier)
## e-EnergieTEC - Guide Rapide NIGELEC

Ce manuel est destiné aux agents commerciaux et vendeurs aux guichets NIGELEC chargés de générer des crédits STS (Tokens) pour les clients.

---

## 1. Ouverture de Caisse (Shift)
À votre arrivée au guichet :
1. Connectez-vous à l'application avec votre identifiant et mot de passe.
2. Le système vous demandera de **Démarrer un Quart (Shift)**.
3. Entrez le fond de caisse initial (les espèces dont vous disposez) et validez. Votre caisse est maintenant ouverte.

## 2. Rechercher un Compteur / Client
Lorsqu'un client se présente pour acheter du crédit :
1. Allez dans le menu **Vente STS (Prepaid)**.
2. Dans la barre de recherche, entrez le **Numéro du Compteur** (généralement 11 chiffres) ou le **Nom du client**.
3. Cliquez sur la loupe ou appuyez sur Entrée.
4. Les informations du client apparaissent. **Vérifiez toujours le nom affiché à l'écran avec le client** pour vous assurer qu'il recharge le bon compteur.

## 3. Générer un Token (Vente)
Une fois le compteur trouvé :
1. Dans la zone "Montant", saisissez la somme payée par le client en **FCFA** (Ex: 5000).
2. Sélectionnez le canal de paiement (Espèces, Orange, Airtel, NITA).
3. Cliquez sur le bouton **Générer Token**.
4. Le système calcule automatiquement l'énergie (kWh) en fonction de la tranche tarifaire du client, soustrait les frais fixes éventuels, et génère le code STS à 20 chiffres.

## 4. Impression du Reçu
Dès que le token est généré, une fenêtre avec le reçu s'affiche.
1. Cliquez sur le bouton **Imprimer (Thermique)** en haut du reçu.
2. Le reçu sera envoyé à l'imprimante connectée (format ticket de caisse).
3. Remettez le ticket au client. Le code STS de 20 chiffres y est imprimé en gros caractères.

## 5. Cas Particuliers
*   **Compteur en "Alerte Fraude" :** Si le compteur est bloqué suite à une tentative d'effraction (Tamper), le système refusera la vente. Vous devez orienter le client vers le Centre de Contrôle pour une levée d'alarme.
*   **Erreur de Frappe (Montant) :** Une fois le token généré, il est **impossible de l'annuler** ou de rembourser directement depuis votre profil. Tout litige doit faire l'objet de l'ouverture d'un Ticket technique.

## 6. Clôture de Caisse (Fin de journée)
À la fin de votre service :
1. Cliquez sur l'icône de votre profil en haut à droite.
2. Sélectionnez **Clôturer le Quart**.
3. Un récapitulatif de toutes vos ventes de la journée (en espèces et digital) s'affiche.
4. Confirmez pour clôturer. Vous pouvez alors imprimer le rapport de fermeture pour la comptabilité de l'agence.
